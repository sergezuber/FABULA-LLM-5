# Agent rules (FABULA-LLM-5)

- **Search before re-deriving.** Before recomputing or re-discovering a fact, a decision, or how something was done, call `session_search` FIRST — prior sessions are indexed and may already hold it. Don't rebuild from scratch what you can recall.
- **Write durable gotchas down.** When you learn a non-obvious, durable fact (a build quirk, a path, a fix that worked), record it rather than rediscovering it next session. Text outlives a session; "mental notes" don't.
